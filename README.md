# NovaBuy Fashion

**NovaBuy Fashion** is a modern, scalable e-commerce web application built with Django. It offers a
seamless online shopping experience focused on fashion products with user-friendly navigation,
product browsing, cart management, and admin features.

## Features

- User registration and authentication
- Product listing by categories
- Search functionality
- Shopping cart and order management
- Admin panel for product and order management
- Responsive design for desktop and mobile devices

## Technologies Used

- Python 3.x
- Django Web Framework
- SQLite (default, can be switched to other DBs)
- HTML5, CSS3, Bootstrap (for responsive UI)
- JavaScript (basic interactions)
